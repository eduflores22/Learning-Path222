Intorducción al lenguaje de marcado HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Proporciona la estructura y el contenido de una página web, definiendo los elementos y etiquetas que se utilizan para describir el formato y la presentación de los documentos en línea. HTML se compone de una serie de etiquetas o elementos, cada uno con una función específica. Estas etiquetas se utilizan para marcar y estructurar diferentes partes del contenido de una página web, como encabezados, párrafos, imágenes, enlaces, listas, tablas, formularios y otros elementos. Para escribir en la versión actual de html (html 5) debemos colocar la declaración !DOCTYPE html (entre signos de menor y mallor).

Elementos

Son las partes fundamentales que componen una página HTML. Están representados por etiquetas y se utilizan para estructurar y dar formato al contenido. Cada elemento comienza con una etiqueta de apertura etiqueta (entre signos de mayor y menor) y termina con una etiqueta de cierre /etiqueta (entre signos de menor y mayor).

Atributos

Proporcionan información adicional sobre los elementos HTML. Se agregan dentro de las etiquetas de apertura y se componen de un nombre y un valor. Los atributos ayudan a controlar el comportamiento o la apariencia de los elementos. El atributo se escribe dentro de la etiqueta del elemento seguido de un = y “” con el valor del atributo. Puedes incluir varios atributos separados por espacios en la etiqueta de apertura del elemento.

Intorducción

CSS significa Cascading Style Sheets, significan “Hojas de estilo en cascada”, se llamada de cascada porque se lee, procesa y aplica el código de arriba abajo. Teniendo en cuenta que las páginas web es realmente un documento de texto, la base de este documento el lenguaje HTML, uniendo el HTML con CSS permite darle forma, color, posición y muchas más características de forma automática y masiva, durante el ensayo a un documento web. Utilizamos CSS para separar el contenido de la presentación, con el fin de que a largo plazo el código sea más facil de entender y modificar en el caso de que fuese necesario.

Recordatorio:

  • Documentos HTML (contenido) = solo información y datos.
  • Documentos CSS (presentación) = solo aspectos relacionados con el estilo

Sintaxis

Selector: elemento o elementos del documento que vamos a seleccionar para aplicarle un estilo concreto.

Propiedad: es la característica principal que vamos a definir con el valor indicado a continuación.

Valor: cada propiedad tiene una serie de valores concretos asignables, que cambian el comportamiento.

Comentario: Un trozo de texto “normal” ente /* comentario */, no los muestra el navegador a los usuarios, solo a los usuarios que visualicen el código.

Regla: par de propiedad con su respectivo valor asociado.

Id y Class

Los selectores de ID y class son formas de identificar elementos en nuestra HTML sobre los que queremos aplicar estilos.

Los selectores de ID se definen utilizando en simbolo del hashtag “#” seguido del nombre del ID. id= “listaa”.

A diferencia de los ID, las clases pueden ser utilizadas en múltiples elementos. Para aplicar un selector de clase a un elemento de nuestra hoja html agregando el atributo “class” Los selectores de clase se definen utilizando un . seguido del nombre que le adjudiquemos a la clase. Ejemplo: class= “nombre de la clase”.

También, es posible asignar un ID y una clase a un mismo elemento HTML, esto permite combinar las ventajas de ambos selectores. (estilos específicos utilizando el ID y estilos comunes utilizando la clase).

Intorducción

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es ampliamente utilizado en el desarrollo web para agregar interactividad, dinamismo y funcionalidades a los sitios y aplicaciones.

Como lenguaje de programación, JavaScript permite a los desarrolladores controlar el comportamiento de los elementos HTML, manipular datos, responder a eventos y realizar cálculos complejos. Es compatible con todos los navegadores web modernos y se ejecuta en el lado del cliente, lo que significa que se ejecuta en el navegador del usuario sin necesidad de comunicarse constantemente con el servidor.

JavaScript es conocido por su versatilidad y flexibilidad. Puede utilizarse para crear desde pequeños scripts que realizan tareas específicas hasta aplicaciones web completas y robustas. También es compatible con múltiples paradigmas de programación, como programación orientada a objetos, programación funcional y programación basada en eventos.

Aplicación

En la aplicación se utiliza el método addEventListener para asignar un evento de clic al elemento con el id "contexto-btn". Cuando se hace clic en ese elemento, se ejecuta la función proporcionada como segundo argumento.

Dentro de la función, se busca el elemento con el id "contexto-container" utilizando document.getElementById("contexto-container") y se asigna a la variable container.

Luego, se verifica el valor de la propiedad display del estilo del elemento container usando container.style.display. Si el valor es "none" (es decir, el elemento está oculto), se cambia el valor de display a "block" para mostrar el elemento estableciendo container.style.display = "block". Si el valor es diferente de "none" (es decir, el elemento está visible), se cambia el valor de display a "none" para ocultar el elemento estableciendo container.style.display = "none".

Menú principal

En la parte superior derecha elegimos el archivo sobre el que queremos trabajar, estos se van abriendo en la parte superior media y haremos click en la que queremos trabajar. También se puede configurar para tener uno o más archivos en la misma vista, para esto utilizaremos el menú lateral derecho.

Con el menú lateral izquierdo en primer lugar se encuentra archivo (File): Contiene comandos relacionados con la gestión de archivos y proyectos, como abrir y guardar archivos, cambiar de carpeta de trabajo y opciones de impresión. Edición (Edit): Proporciona comandos para realizar acciones de edición de texto, como cortar, copiar, pegar, deshacer y rehacer. También incluye opciones de búsqueda y reemplazo. Ver (View): Este menú ofrece opciones para personalizar la apariencia y el diseño de VS Code. Aquí puedes acceder a comandos para cambiar el diseño de la interfaz, mostrar u ocultar paneles y barras de herramientas, y ajustar la configuración de zoom. Buscar (Find): Proporciona opciones y comandos para buscar texto en archivos abiertos y en todo el proyecto, así como para realizar reemplazos.
